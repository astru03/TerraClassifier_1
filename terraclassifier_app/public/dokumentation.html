Dokumentation

Thank you for choosing to use the Terra Classifier.
To give you the best possible experience with our product, we have put together an overview of all the buttons, including how they work and operating instructions.
The application is structured according to the principle of a one-way street. There are eight buttons on the right-hand side. For a successful classification, the first six buttons must be processed. As the order must be strictly adhered to (one-way street), the buttons that have not yet been processed are grayed out and cannot be selected. At the same time, you can view your current processing step at any time. In the event of problems, you can use this documentation to search specifically for your current processing step and rectify possible application errors more quickly.
The following table is sorted according to the sequence of the application and not according to the layout. This also simplifies troubleshooting and error prevention.

Sentinal-2 Mit diesem Button lösen Sie den Prozess der Satellitenbildbeschaffung aus. Bevor Sie diesen Button betätigen können, müssen Sie zuerst ein Bereich auswählen, für den ein Satellitenbild geladen werden soll. Um Maschinelles Lernen für die Klassifikation nutzen zu können, braucht das Programm ein Satellitenbild, das ausgewertet werden kann. Mit dem Rechteck-Button am linken Kartenrand, können Sie das gewünschte Gebiet einzeichnen. Nachdem Sie nun den Satelliten-Knopf betätigt haben, werden Sie aufgefordert ein Datum auszuwählen, aus diesem das Satellitenbild stammen soll. Sobald Sie Ihr Datum innerhalb des Kalenderwidgets und einen Wert für die Wolkenbedeckung ausgewählt haben, bestätigen Sie mit „OK“. Das Satellitenbild wird nun geladen.
Training data Für eine erfolgreiche Klassifikation, werden Trainingsdaten benötigt. Diese Trainingsdaten können Sie entweder per Upload im GeoJSON- oder GeoPackage-Format hochladen oder gleich hier in der Webanwendung erstellen. Dabei ist sicherzustellen, dass die Trainingsdaten nur innerhalb des vorher gewählten Area Of Training (AOT) liegen. Zeichnen Sie mit dem Rechteck-Button am linken Kartenrand ein AOT. Lösen Sie nun den Trainingsdaten-Button aus und wählen eine der beiden Optionen aus.
Upload: Nachdem sich das Fenster für den Upload geöffnet hat, suchen und wählen Sie Ihr Verzeichnis, in dem sich die Datei befindet.
Erstellen: Haben Sie sich für das Erstellen neuer Trainingsdaten innerhalb des Browsers entschieden, erscheint mittig am linken Rand der Karte ein neues Polygon-Symbol (Icon). Nach dem Klicken auf dasselbige, ist es nun möglich neue Trainingsdaten zu erstellen. Nach jedem erstellten Polygon werden Sie aufgefordert, eine passende Objekt-ID zu vergeben.
Wenn Sie mit dem Upload oder der Erstellung fertig sind, können Sie zum nächsten Bearbeitungsschritt übergehen, indem Sie auf der rechten Seite das Zahnrad-Symbol anklicken.
Algorithm Nach dem Klicken auf dieses Symbol, können Sie zwischen zwei verschiedenen Algorithmen wählen. Zur Auswahl stehen der „Minimum Distance“- und der „Random Forest“-Algorithmus. Bestätigen Sie Ihre Wahl mit „OK“. Den nächsten Bearbeitungsschritt starten Sie durch Klicken des Positions-Symbols (Icon).
AOI Nachdem Sie die Schaltfläche betätigt haben, wird die Area Of Training unsichtbar. Nun können Sie durch Klicken auf das Rechteck-Symbol (Icon) eine Area Of Interest einzeichnen. Dies stellt den zu klassifizierenden Bereich dar. Ist Ihr AOI eingezeichnet, fahren sie mit dem nächsten Bearbeitungsschritt fort (Icon).

Classification Mit Klick auf diesen Button werden Sie aufgefordert eine Auflösung (Meter pro Pixel) auszuwählen. Sie können zwischen drei Optionen wählen. Mit „OK“ bestätigen Sie Ihre Auswahl und starten gleichzeitig die Berechnung. Dies kann einige Sekunden in Anspruch nehmen. Sobald die Berechnung beendet ist, werden Sie durch ein Pop-Up gefragt, ob Sie die Klassifikation als GeoTiff herunterladen möchten. Nachdem Sie Ihre Wahl getroffen haben, wird die Klassifikation nach wenigen Sekunden in die Karte geladen.

Setup (lokal)

Schritt Anweisung
1. Klonen Sie das folgende GitHub-Repository in eine IDE (zum Beispiel VS Code).
2. Navigieren Sie im Terminal mit dem Befehl "cd terraclassifier_app" in das gleichnamige Verzeichnis.
3. Nutzen Sie den Befehl "npm install", um alle notwenidgen Module und Komponenten zu installieren. Die Installation kann bis zu zwei Minuten in Anspruch nehmen.
4. Nachdem die Installation erfolgreich abgeschlossen wurde, können Sie die Anwendung starten. Geben Sie dafür den Befehl "npm start" ein. Nach wenigen Sekunden ist die Anwendung im Browser unter der URL: http://localhost:3000 abrufbar.
